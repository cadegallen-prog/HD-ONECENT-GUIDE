Screen Reader Navigation

For Screen Reader Users

This site is optimized for screen readers with proper ARIA labels, landmark regions, and semantic HTML.

  • Landmarks: Use your screen reader's landmark navigation to jump between sections (navigation, main content, footer)
  • Headings: All pages use proper heading hierarchy (H1 ‚Üí H2 ‚Üí H3) for easy navigation
  • Live Regions: Progress updates, search results, and notifications are announced automatically
  • Skip Links: Press Tab at page load to reveal "Skip to main content" link
