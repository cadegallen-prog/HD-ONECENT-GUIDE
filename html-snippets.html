HTML Snippets & Component Examples

This page contains all the HTML markup needed to integrate the interactive features.

1. Basic Page Template

Use this template for all pages with interactive features:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Page Title - Home Depot Penny Items Guide</title> <!-- Styles --> <link rel="stylesheet" href="styles.css"> <link rel="stylesheet" href="component-styles.css"> </head> <body> <nav> <!-- Navigation links --> </nav> <!-- Breadcrumb navigation --> <div id="breadcrumb-nav"></div> <main> <h1>Page Heading</h1> <!-- Page content --> <!-- Suggested next pages --> <div id="suggested-next"></div> </main> <footer> <!-- Footer content --> </footer> <!-- Scripts --> <script src="base.js"></script> <script src="learning-platform.js"></script> <script src="ui-components.js"></script> <script src="search.js"></script> <!-- Initialize --> <script> document.addEventListener('DOMContentLoaded', () => { if (window.initUIComponents) window.initUIComponents(); if (window.initSearch) window.initSearch(); }); </script> </body> </html>

4. Modal System

// Confirmation modal window.UIComponents.Modal.open({ title: 'Confirm Action', content: '<p>Are you sure you want to proceed?</p>', type: 'confirm', confirmText: 'Yes, Continue', cancelText: 'Cancel', onConfirm: () => { console.log('User confirmed'); // Your action here }, onCancel: () => { console.log('User cancelled'); } }); // Info modal (just OK button) window.UIComponents.Modal.open({ title: 'Information', content: '<p>Here is some important information.</p>', type: 'info', confirmText: 'Got it' }); // Close modal programmatically window.UIComponents.Modal.close();

5. Toast Notifications

// Success toast window.UIComponents.Toast.success('Changes saved successfully!'); // Error toast window.UIComponents.Toast.error('Failed to save changes'); // Warning toast window.UIComponents.Toast.warning('Please review your input'); // Info toast window.UIComponents.Toast.info('Did you know: you can use Ctrl+K to search'); // Custom duration (default is 3000ms) window.UIComponents.Toast.success('This will show for 5 seconds', 5000); // Manual dismiss (duration = 0) window.UIComponents.Toast.info('Click X to dismiss', 0);

6. Loading States

JavaScript API for loading indicators:

// Show loading on specific element window.UIComponents.Loading.show('#my-element', 'Loading data...'); // Hide loading window.UIComponents.Loading.hide('#my-element'); // Full-page loading overlay window.UIComponents.Loading.overlay(true, 'Processing...'); // Hide overlay window.UIComponents.Loading.overlay(false); // Example usage with async operation async function loadData() { window.UIComponents.Loading.show('#content', 'Loading...'); try { const data = await fetch('/api/data'); // Process data } catch (error) { console.error(error); } finally { window.UIComponents.Loading.hide('#content'); } }

12. Quick Reference

Essential Files

  • component-styles.css - All component styles
  • ui-components.js - Breadcrumbs, tooltips, modals, toasts, loading, errors
  • search.js + search-index.json - Search system
  • quiz-system.js + quizzes.json - Quiz system

Initialization Order

  1. Load CSS files
  2. Load existing scripts (base.js, learning-platform.js)
  3. Load new feature scripts (ui-components.js, search.js, quiz-system.js)
  4. Call init functions in DOMContentLoaded

Required HTML Elements

  • <div id="breadcrumb-nav"></div> - For breadcrumbs
  • <div id="suggested-next"></div> - For suggested pages
  • <div id="quiz-container"></div> - For quizzes (quizzes.html only)

Keyboard Shortcuts

  • Ctrl+K or Cmd+K - Open search
  • / - Open search (when not in input)
  • ↑ ↓ - Navigate search results
  • Enter - Select search result
  • Esc - Close search/modals
