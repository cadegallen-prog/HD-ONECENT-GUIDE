Technical Notes

Site Architecture

This site is intentionally simple:

  • Pure HTML/CSS/JavaScript - No frameworks or dependencies
  • Static files only - No backend, database, or server-side processing
  • Self-contained - All resources hosted locally, no CDN dependencies
  • Fast and lightweight - Optimized for quick loading on any device

Why This Approach?

  • Accessibility: Works on any browser, any device, even offline
  • Maintainability: Easy to update without complex build processes
  • Reliability: No external dependencies that can break
  • Privacy: No tracking, analytics, or data collection
  • Longevity: Will work for years without updates

  • Check the home page for links to new resources
