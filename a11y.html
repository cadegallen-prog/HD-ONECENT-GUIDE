Accessibility Tools & Options

3 min read All Users

Our Commitment to Accessibility

This guide is designed to be accessible to everyone, regardless of ability. We strive to meet or exceed WCAG 2.1 AA standards and provide multiple ways to customize your reading experience.

Quick access: Press Alt + A to open the floating accessibility widget from any page.

Visual Accessibility Options

High Contrast Mode

Increase contrast for better readability, with black backgrounds and white text.

Auto-detects system preference

Color Blind Modes

Adjust colors and add patterns to accommodate different types of color blindness.

Dyslexia-Friendly Mode

Increase letter spacing, line height, and use an off-white background for easier reading.

Font Size

Adjust text size to your preference.

Browser zoom (Ctrl +/-) also works

Focus Indicators

All focusable elements have visible 3px focus outlines with 3:1 contrast ratio.

Always enabled

Motion & Animation Controls

Reduce Motion

Disable or reduce all animations and transitions for users with vestibular disorders.

Auto-detects prefers-reduced-motion

Animation Speed

Control the speed of animations on the site.

Reading Assistance

Text-to-Speech

Have page content read aloud using your browser's built-in speech synthesis.

1.0x
Highlights words as they're spoken

Focus Mode

Minimize visual distractions by dimming everything except the main content.

Keyboard Navigation

Fully Keyboard Accessible

This entire site can be navigated without a mouse using keyboard shortcuts.

  • Tab / Shift+Tab: Navigate between focusable elements
  • Enter / Space: Activate buttons and links
  • Arrow keys: Navigate FAQ items and page navigation
  • Esc: Close modals and search
  • N / P: Next / Previous page
  • / or Ctrl+K: Focus search

Heading Hierarchy

Proper Document Outline

All pages use proper heading hierarchy for easy navigation:

  • H1: One per page (page title)
  • H2: Major sections
  • H3: Subsections
  • No skipped levels (e.g., H1 ‚Üí H3)

Screen reader users can navigate by headings using their assistive technology's heading navigation features.
